CSS Grid Notes

what is css grid ?

The CSS grid is a newer standard that makes it easy to build complex responsive layouts. It works by turning an HTML element into a grid, and lets you place child elements anywhere within.

create a css grid

Turn any html element into a grid by setting its display propert to grid. the syntax for doing so is:
.container {
display: grid;
grid-template-columns: 50px 50px 20px;
grid-template-rows: numbers; }
This would set up a grid with 3 columns, 2 are 50px wide and one 20px. ditto for the rows.
We can also use other units than pixels like: fr: which sets the column or row to a fraction of the available space, auto, which set it automatically, and %, which adjusts the column or row to the percenth width of its container.

create a column / row gap

without this, the columns/rows would be tight up against each other, to add a gap between the elements we use: grid-column/row-gap: size;
or just use the short hand grid-gap: size1 size2; (1 for rows, 2 for columns)

grid-row and grid-column

they are properties of the itmes not the container, they define how many rows/columns the item will use, the syntax is like: grid-row/column: starting line / ending line the following example is for a 4*4 grid which got 3 row and colum lines

grid-column: 1/3
grid-row: 1/3
grid-column: 3/4
grid-row: 1/3
grid-column: 1/4
grid-row: 3/4

aligning an item with justify-self

the content of each item in a css grid is located in a cell,we can align the content's position inside a cell horizontally with justify-self property on a grid item, by default, this prop has the stretch value, which makes the content fit the whole cell.
align-self is used to align items vertically, it takes the same values as justify.
possible values:

  1. start: align content at the left of the cell
  2. end: align content at the right of the cell
  3. center: align content at the right of the cell

align all items of a grid

in the previous section we aligned horizontally and vertically specific items of a grid, we have the possibility to align all of them with justify-elements and align-elements, they take the same values as justify/align-self (center, stretch by default, start, end)

Divide the Grid Into an Area Template

we can group cells of the grid together into an area and give it a custom name. we do this using grid-template-areas on the container like this:
grid-template-areas: "truc truc truc" "machin machin garbage" "footer footer footer"
this merges the top 3 cells and gives them the name truc, merges the 2 first cells in the 2nd row and gives them the name machin, gives the 3rd cell of the 2nd row the name garbage, and gives the last 3 cells the name footer.
every word represents a cell and every pair of quotation marks representats a row.
we can use a dot(.) to designate an empty cell in the grid.

place an item in a grid area

use the grid-area property, the value is the name of area.
example: .item { grid-area: footer; }
the grid-area property can also be used with areas without given names like this:
grid-area: horizontal line to start at / vertical line to start at / horizontal line to end at / vertical line to end at;

avoid repetition using the repeat function

instead of using grid-template-rows: 10px 10px 10px... for example, we can use this grid-template-rows: repeat(n,10px) which would give a grid with n rows of size 10px, other variants of this function could be used, for instance: grid-template-rows: repeat(2,1fr 10px) 15px is equivalent to grid-template-rows: 1fr 10px 1fr 10px 15px
We can put auto-fill instead of a specific number, it inserts as many rows or columns depending on the size of the container, When the container changes size, it keeps inserting columns/rows and stretching them until it can insert another one. Note: If your container can't fit all your items on one row, it will move them down to a new one
auto-fit works almost identically as auto-fill. the only difference is that when the container's size exceeds the size of all the items combined, auto-fill keeps inserting empty rows/columns and pushes items to the side, while auto-fit collapses those empty rows/columns and stretches the items to fit the size of the container.

the minmax function

the minmax function is used to give a minimum and a maximum value that a row/column size could take, for example, grid-template-columns: 100px minmax(50px, 200px); creates 2 columns, one of size 100px, the second has the minimum of 50 and the max of 200px, depending on the size of the container.
